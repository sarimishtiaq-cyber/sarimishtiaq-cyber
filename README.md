My work sits at the intersection of **AI Engineering**, **Revenue Operations**, **Automation**, **CRM Architecture**, **Data Engineering**, and **Business Intelligence**.

## How I Work

- **Systems before code.** Design the architecture, data flow, and edge cases before implementation.
- **Automation first.** Eliminate repetitive work with reliable workflows and AI-powered processes.
- **Business-driven engineering.** Every technical decision should create measurable business value.
- **Keep it lean.** Simple, maintainable solutions outperform unnecessary complexity.
- **Build for scale.** Internal tools should be reliable, extensible, and easy for small teams to maintain.
